Integrate Nyx Uploader into your own application.

Authentication

The public upload endpoint does not require authentication. You can make requests directly without an API key.

Upload Endpoint

To upload a file, send a POST request with multipart/form-data to the following endpoint:

POST /upload

Parameters

  • file (form-data, required): The file you want to upload.

Example: cURL

curl -X POST "https://uploader.obscuraworks.com/upload" \
-F "file=@/path/to/your/image.jpg"

Example: Node.js (with Axios)

const axios = require('axios');
const fs = require('fs');
const FormData = require('form-data');

async function uploadFile(filePath) {
  const url = 'https://uploader.obscuraworks.com/upload';
  const form = new FormData();
  form.append('file', fs.createReadStream(filePath));

  try {
    const response = await axios.post(url, form, {
      headers: {
        ...form.getHeaders(),
      },
    });
    console.log('File uploaded successfully:', response.data);
  } catch (error) {
    console.error('Error uploading file:', error.response.data);
  }
}

uploadFile('/path/to/your/image.jpg');

Example: Python (with Requests)

import requests

url = "https://uploader.obscuraworks.com/upload"
file_path = "/path/to/your/image.jpg"

with open(file_path, "rb") as f:
    files = {"file": (f.name, f, "image/jpeg")}
    response = requests.post(url, files=files)

if response.status_code == 200:
    print("File uploaded successfully:")
    print(response.json())
else:
    print(f"Error uploading file: {response.status_code}")
    print(response.json())
